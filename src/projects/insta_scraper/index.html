Instagram Profile Scraper

Este é um projeto que fiz para colocar em prática algumas coisas que aprendi sobre APIs.

Na criação deste projeto utilizei uma API disponibilizada no site da RapidAPI e alguns conhecimentos básicos em Javascript, além de HTML Semântico e CSS.

Com ela conseguimos coletar dados como nome de usuário, biografia, ID, seguidores e muitas outras coisas!
