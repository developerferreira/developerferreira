C-Combos

Neste projeto, o usuário pode criar as melhores combinações de luvas e facas. Conta com a possibilidade de filtrar por preços, cores, tipos de itens e preço. Além de receber apenas recomendações que combine com a faca/luva de sua escolha.

Para a criação deste projeto, foi utilizado HTML, CSS e Javascript. Para as imagens, utilizei o aplicativo Photoshop para editar cada luva e faca disponiveis no projeto, uma vez que a empresa responsável pelo jogo não possui uma API pública. O projeto conta também com uma excelente responsividade, sendo compátivel com vários dispositivos, garantindo uma ótima experiência ao usuário. Além disso, para o armazenamento de informações de cada item usei JSON. Dentro dele existem atributos como: preço de cada luva e faca disponível, uma breve descrição do item e imagens de cada um deles e as cores.
