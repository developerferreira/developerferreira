C-Combos

Neste projeto, o usuário pode criar os melhores kits com base em sua quantidade atual de moedas dentro do jogo. Além de receber apenas recomendações que combine com sua faca/luva.

Para a criação deste projeto usei HTML Semântico, CSS e Javascript. Para as imagens, utilizei o aplicativo "Photoshop" para editar cada luva/faca disponiveis no projeto, uma vez que a empresa responsável pelo jogo não possui uma API pública.
